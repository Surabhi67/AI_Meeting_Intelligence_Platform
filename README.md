# AI Meeting Summarizer

AI Meeting Summarizer is a full-stack web application that allows users to upload meeting audio, generate transcripts and AI summaries, and share meeting notes with other participants.

The project is built with:

- React (frontend)
- FastAPI (backend)
- PostgreSQL
- SQLAlchemy
- Docker

## Features

- User authentication
- Audio upload support
- Meeting dashboard
- Transcript generation
- AI-powered summaries
- Shared meeting access
- PostgreSQL database integration
